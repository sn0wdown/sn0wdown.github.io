hexo + github 搭建个人blog

发表于 2017-06-05 | 分类于 基础教程 | 阅读次数

现在越来越多的人喜欢利用Github搭建静态网站，原因不外乎简单省钱。本人也利用hexo+github搭建了本博客，用于分享一些心得。在此详细记录下我是如何利用hexo+github搭建静态博客以及一些配置相关问题，以免过后遗忘，且当备份之用。

准备工作

下载node.js并安装（官网下载安装），默认会安装npm。
下载安装git（官网下载安装）
下载安装hexo。方法：打开cmd或git 运行  
$ npm install -g hexo-cli

本地搭建hexo静态博客

安装 Hexo 完成后，请执行下列命令，Hexo 将会在指定文件夹中新建所需要的文件。 

$ hexo init <folder>  
$ cd <folder>  
$ npm install

将博客与Github关联

在Github上创建名字为XXX.github.io的项目，XXX为自己的github用户名,添加SSH公钥到Account settings -> SSH Keys -> Add SSH Key,打开本地的项目内的_config.yml配置文件，找d到deploy字段,将其中的type设置为git

deploy:
  type: git
  repository: https://github.com/yourname/yourname.github.io.git
  branch: master

运行：npm install hexo-deployer-git –save
运行：hexo g（本地生成静态文件）
运行：hexo d（将本地静态文件推送至Github）

此时，打开浏览器，访问http://yourname.github.io

更新文章

在项目目录下执行：hexo new “我的第一篇文章”，会在source->_posts文件夹内生成一个.md文件。编辑该文件（遵循Markdown规则）修改起始字段 

title 文章的标题
date 创建日期 （文件的创建日期 ）
tags 标签
categories 分类
编写正文内容（MakeDown）  

编写完，执行即可部署到github
$ hexo clean
$ hexo d -g

其它的配置

加入阅读次数统计

在next主题里，已经引入了leancloud用于统计阅读次数，我们只需要启用即可，我们先要在leancloud网站注册创建帐户，记录我们创建的app_id和app_key，然后在主题配置文件中搜索leancloud,填写我们对应的值即可 

leancloud_visitors:  
  enable: true  
  app_id: tenS5cAVxxxxxxxxxVrVD-gzGzoHsz  
  app_key: Sme2xxxxxxxxxxxxxxBG

添加友链

在主题配置文件中搜索link,按如下格式添加即可

links_title: 友情链接  
links_layout: block  
links_layout: inline  
links:
  aptx's blog: https://aptx0.github.io/  
  Urahara's blog: http://reverse-tcp.xyz/

添加其它社区链接的话，找到social字段添加对应链接即可 

social: 
  freebuf: http://www.freebuf.com
  先知社区: https://xianzhi.aliyun.com/forum/

自定义new命令新建文章模板

执行new命令，生成指定名称的文章至hexo\source\_posts\postName.md

hexo new [layout] "postName" #新建文章

其中layout是可选参数，默认值为post。有哪些layout呢，请到scaffolds目录下查看，这些文件名称就是layout名称。当然你可以添加自己的layout，方法就是添加一个文件即可，同时你也可以编辑现有的layout，比如post的layout默认是hexo\scaffolds\post.md

生成带照片的文章：
新建hexo\scaffolds\photo.md文件

---
layout: {{ layout }}
title: {{ title }}
date: {{ date }}
tags: 
categories: 
photos: 
-
---

然后每次可以执行带layout的new命令生成照片文章：

hexo new photo "photoPostName" #新建照片文章

点击出现心形

将 love.js 文件添加到 \themes\next\source\js\src 文件目录下。
找到 \themes\next\layout_layout.swing 文件， 在文件的后面， 标签之前 添加以下代码：

<!-- 页面点击小红心 -->
<script type="text/javascript" src="/js/src/love.js"></script>

图床

这里使用的是七牛云存储,在七牛网站注册张号,建立存储空间,然后可以使用MPic工具,配置我们创建的AK和SK，即可直接使用，复制的链接就是markdown的图片格式

一些常用的命令

hexo new "postName" #新建文章  
hexo new page "pageName" #新建页面  
hexo generate #生成静态页面至public目录  
hexo server #开启预览访问端口（默认端口4000，'ctrl + c'关闭server）  
hexo deploy #将.deploy目录部署到GitHub  
hexo help  #查看帮助    
hexo version  #查看Hexo的版本

Q&A

问：如何让文章想只显示一部分和一个 阅读全文 的按钮？   
答：在文章中加一个 <!--more--> ， <!--more--> 后面的内容就不会显示出来了。
